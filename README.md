# 🎮 Game Zone

A simple and fun browser-based mini game collection built using **HTML, CSS, and JavaScript**.

## 🚀 Features

* ✂️ **Rock Paper Scissors**

  * Play against the computer
  * Score tracking (stored using localStorage)
  * Auto-play mode

* 🔢 **Number Guessing Game**

  * Guess a number between 1–100
  * 10 attempts limit
  * Feedback for high/low guesses

* ⭕ **Tic Tac Toe**

  * 2-player game
  * Score tracking (X, O, Ties)
  * Reset and new game options
